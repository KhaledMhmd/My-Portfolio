MY PORTFOLIO

Bankist App

An e-banking website that lets the user handle their banking transactions that I had as a project in a Javascript course that I was taking. In this website, we have a mini database for multiple users, each has their own username, password, account balance, ..etc. The user can request loans from the bank, transfer money to any other user of the website or close their account. It's a demo website of course.

Bankist App (New Interface)

A new interface for the previously mentioned bankist website, which was also a project in a course. This new interface has a more marketing approach for the bank, a prettier looking landing page which has tapped components, sliders, fading in and out of sections, lazy loading images and much more, all while preserving the minimalistic identity of the website.

Mapty App

A health/fitness website that lets users keep track of their workouts, document them and store them in the website. The user pinpoints the location of the workout on a map using geolocation Api and leaflet library to display map at the user's current location, and storing these workouts in local storage where we can also load previous workouts from. This was also a project in a course I took.
